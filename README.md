# Praveno — Live Demo

**Try it:** https://ipaja80.github.io/praveno-web/

Secure business workflow app (demo). Sign in with any role (Owner / Manager / Member / Approver / Viewer)
to explore role-based access. All data stays in your browser — nothing is sent to a server.

- **iPhone/iPad:** open in Safari → Share → **Add to Home Screen** to install as an app
- **Android:** Chrome will offer "Install app" (or use the APK from the private praveno-app repo)

Built with a NIST SP 800-37/800-53 + OWASP ASVS-aligned design — see the in-app **Governance** tab.
